numbers, 숫자 타입

  • integer, 딱 떨어지는 수 -1, 0, 1, 2
  • float, 소수점 이하를 포함하는 수 3.14, -9.81;

string, 여러개의 문자들

  • const name = "이서희";
  • const starSign = "전갈자리'

typeof, 타입 확인하는 법!

  • typeof 1234;
  • typeof "1234";
  • typeof starSign;

boolean, 참과 거짓!

  • typeof true;
  • typeof false;
  • const hungry = true;
  • typeof hungry;

미지정(undefined) vs 값이 없음(null)

  • let dinner; // undefined, 이름표만 만들고 값을 미지정!
  • typeof dinner
  • dinner = "피자"; // string 값을 변수에 연결
  • typeof dinner
  • dinner = null; // null, 값을 연결하지 않기로 함!
  • typeof dinner
